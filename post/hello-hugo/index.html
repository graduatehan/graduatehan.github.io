1. HUGO 安装

官网：https://gohugo.io/installation/windows/

下述已 Windows 为例，其他系统类似。

1.1 版本

有两个版本：standard and extended

扩展版（extended）功能更强大，推荐安装。

1.2 安装方式

  1. 二进制安装包（推荐，最简单）
    1. 下载地址：https://github.com/gohugoio/hugo/releases/latest
    2. 根据操作系统选择对应版本即可，如 hugo_extended_0.120.4_windows-amd64.zip
  2. 包管理器安装，例如 Winget，略
  3. 源码编译，略

2. 使用前的准备工作

2.1 Hugo 添加环境变量

下载安装包解压即可使用，将解压路径添加到环境变量。例如：D:\hugo_extended_0.120.4_windows-amd64

2.2 Git 安装及设置

2.2.1 下载安装

  • 官网下载 https://git-scm.com/downloads
  • 使用包管理器下载

2.2.2 配置

# 配置用户名以及邮箱
git config --global user.name "Your Name"
git config --global user.email "邮箱"
# 查看是否配置成功
git config --global --list

# 生成密钥（默认生成 ~/.ssh 目录下），用于免密连接 github
ssh-keygen -t rsa -C "邮箱"
# 登录github ->进入设置->SSH and GPG key ->新建密钥，将id_rsa.pub内容复制并保存
# 测试连接
ssh -T git@github.com

2.3 提醒

如果使用 Windows :

  1. 请勿使用 cmd or Windows PowerShell 执行下面命令。
  2. 可以使用 git bash or PowerShell

注： PowerShell and Windows PowerShell are different applications.

3. 使用 HUGO 创建博客

官网指导文档： https://gohugo.io/getting-started/quick-start/

# 创建站点,指定站点名 my_blog
hugo new site my_blog

# 进入目录
cd my_blog

# 初始化 git
git init

# 选择喜欢的主题，主题下载地址：https://themes.gohugo.io/

# ananke主题，拉取远程仓库主题
git submodule add https://github.com/theNewDynamic/gohugo-theme-ananke.git themes/ananke

# 设置 使用主题 ananke
echo "theme = 'ananke'" >> hugo.toml

# 本地预览效果
hugo server

# 添加文章， 注意必须在 posts 文件夹下
hugo new content posts/myfirstpost.md  # 生成到 content/posts 文件夹下

# 编译生成静态网站  生成 public 文件夹
hugo

4. 部署（托管）到 GitHub

  1. 创建公开仓库，仓库名：用户名.github.io [必须]
  2. 创建 Git 子模块， 将 public 文件夹下内容 push 到 GitHub 仓库
 git submodule add git@github.com:用户名/用户名.github.io.git public
 
 cd public 
 git add .
 git commit -m "first commit"
 git push origin master
  1. 浏览器访问：https://graduatehan.github.io/

5. 另建一个私有 GitHub 仓库：my_blog

将所有内容交给 Git 管理，并 push 到 GitHub 仓库

编辑.gitignore 文件：

resources/*
.hugo_build.lock

.gitmodules 文件：

  • 无需编辑文件，使用 git submodule add 命令添加
[submodule "themes/ananke"]
	path = themes/ananke
	url = https://github.com/theNewDynamic/gohugo-theme-ananke.git
[submodule "public"]
	path = public
	url = git@github.com:用户名/用户名.github.io.git
